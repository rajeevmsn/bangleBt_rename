Instructions

1. Enter the new name for the watch in the input field below

2. Connect to the watch by clicking the "Connect watch" button

3. Select your watch from the list

4. Enter the new name in the input field below (maximum 15 characters, no spaces or special symbols)

5. Click the "Rename" button to change the name of the watch

6. Restart the watch and the new name will be displayed next time you connect

Note: Please use Google Chrome, Microsoft Edge or any other browser that supports web-bluetooth
